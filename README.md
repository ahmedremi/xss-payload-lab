# XSS Payload Lab

A fully static browser security playground for exploring XSS payload contexts, output encoding, sandbox behavior, challenge fixes, and CSP rule effects.

## Stack

- React
- Vite
- Tailwind CSS
- lucide-react

## Scripts

```bash
npm install
npm run dev
npm run build
```

The app does not require a backend, database, login system, or serverless API.
